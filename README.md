# Examen Práctico - Análisis y Diseño de Algoritmos

## Descripción

Este proyecto contiene la solución de dos problemas de HackerRank desarrollados en Java 15:

1. Sherlock and the Valid String
2. Max Min

El programa presenta un menú en consola que permite seleccionar cuál de los dos ejercicios se desea ejecutar.

## Requisitos

- Java 15 o superior.

## Ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/TU_USUARIO/NOMBRE_REPOSITORIO.git
```

2. Abrir el proyecto en un IDE compatible con Java (IntelliJ IDEA, Eclipse o Visual Studio Code).

3. Ejecutar la clase `Main`.

4. Seleccionar el ejercicio que se desea probar desde el menú de la consola.

5. Ingresar los datos solicitados por el programa.

## Estructura del proyecto

```
src/
├── Main.java
├── Sherlock.java

```
